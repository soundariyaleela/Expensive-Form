# 💰 Expensive Form Web Application

## 📌 Overview

The **Expensive Form** is a full-stack web application designed to collect and manage user input efficiently. It includes a frontend interface for user interaction and a backend server to handle data processing.

---

## 🚀 Features

* 📝 User-friendly form interface
* ⚡ Fast and responsive design
* 🔗 Backend integration using Node.js
* 📦 Organized project structure (Frontend + Backend)

---

## 🛠️ Technologies Used

### Frontend:

* HTML
* CSS
* JavaScript

### Backend:

* Node.js
* Express.js

---

## 📂 Project Structure

```
Expensive-Form/
│
├── Frontend/
│   ├── index.html
│   ├── style.css
│   └── index.js
│
├── Backend/
│   ├── server.js
│   ├── package.json
│   └── package-lock.json
```

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/Expensive-Form.git
```

### 2️⃣ Navigate to project folder

```
cd Expensive-Form
```

### 3️⃣ Install backend dependencies

```
cd Backend
npm install
```

### 4️⃣ Run the server

```
node server.js
```
